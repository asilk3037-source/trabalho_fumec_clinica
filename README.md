### Resumo por perfil

**Paciente** — agenda consultas com psicólogos disponíveis, acompanha o histórico de sessões (pendente, confirmado, concluído, cancelado) e edita o próprio perfil e foto.

**Psicólogo(a)** — configura a disponibilidade semanal de horários, acompanha os atendimentos agendados e lê os feedbacks recebidos do supervisor. Contas novas precisam ser ativadas por um supervisor.

**Supervisor(a)** — aprova outros supervisores, ativa/desativa psicólogos, confirma/conclui/cancela agendamentos, envia feedback (nota + observações) para os psicólogos e gerencia as mensagens recebidas pelo Fale Conosco.

A plataforma é totalmente responsiva — todas as funcionalidades também funcionam pelo celular.
